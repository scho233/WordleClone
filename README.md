# WordleClone

WordleClone is a game that is similar to the online game Wordle but with **unlimited attempts**!

This program was developed in C++.

# About

**Features:**
  + Colors from red, green, and yellow with red being invalid letter
  + 5 letter puzzles
  + Diverse library of words
  + _5+ letter words coming soon!_
